# iqiyiMetaScrapy
爬取爱奇艺网站的元数据（以狂飙为例）
# PKU InforOrg IM Class
#北京大学信息组织课程元数据环节


#缺陷：可能存在一些重复输出name的情况和None的情况




## 环境要求

为运行本项目，您需要安装以下软件和库：

- Python 3.6或更高版本
- requests库
- BeautifulSoup库

您可以通过pip命令来安装这些库，例如：

pip install requests beautifulsoup4


## 如何使用

1. 克隆或下载该项目源代码。
2. 进入项目目录，并执行以下命令来启动Python解释器：

python


3. 在Python交互式界面中，运行项目
